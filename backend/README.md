# Backend

Backend for providing UI for credential API interation.

# Build requirements

- rust
- clang or gcc

## Fedora (38 or later)

- gtk4-devel
- libadwaita-devel
- openssl-devel
- blueprint-compiler

## Ubuntu (23.10 or later)

- blueprint-compiler
- libgtk-4-dev
- libadwaita-1-dev
- libssl-dev
